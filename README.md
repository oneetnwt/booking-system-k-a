# K&A Resort Booking System

## Description

This is a booking system for K&A Resort. It is a web application that allows users to book rooms in the resort.

## Features

- User Authentication
- Room Booking
- Payment Integration
- Admin Panel

## Technologies Used

- Node.js
- Express
- MongoDB
- React
- Tailwind CSS  

## Installation

1. Clone the repository
2. Install dependencies
3. Start the server
4. Start the client
